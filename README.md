# Exploratory Data Analysis (EDA) Project

## Objective
Analyze Titanic dataset to uncover patterns and trends.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Features
- Data Cleaning
- Missing Value Handling
- Statistical Analysis
- Correlation Heatmap
- Data Visualization
- Insights Extraction

## Dataset
Titanic Dataset

## Output
Generated graphs and insights about passenger survival patterns.
